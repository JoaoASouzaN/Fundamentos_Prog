# Sintaxe basica do HTML O Html é uma linguagem de marcação e estruturação das paginas web, atualmente ele está na sua quinta versão e em conjunto com o css, JavaScript, React, Node e outras linguagens, sustenta toda a rede web que temos atualmente. Ele funciona como base para a construção do conteúdo, definindo elementos como títulos, parágrafos, imagens, links e seções.
